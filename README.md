# ap-carousel

A simple, functional and fully customizable carousel web component.
content-agnostic
---
## Usage
```html
<ap-carousel>
    <div slot="ap-slide">1</div>
    <div slot="ap-slide">2</div>
    <div slot="ap-slide">3</div>


    <div slot="ap-prev"><</div>
    <div slot="ap-next">></div>
</ap-carousel>
```
